html

ჰიპერტექსტის მარკირების ენა (HTML) არის სტანდარტული მარკირების ენა დოკუმენტებისთვის, რომლებიც შექმნილია ვებ ბრაუზერში გამოსატანად. ის განსაზღვრავს ვებ კონტენტის შინაარსს და სტრუქტურას. მას ხშირად ეხმარება ისეთი ტექნოლოგიები, როგორიცაა (CSS) და სკრიპტირების ენები, როგორიცაა JavaScript.

css

კასკადურ სტილთა ფურცლები (ინგლ. Cascading Style Sheets, CSS) — სტილთა ფურცლების ენა, რომელიც გამოიყენება მონიშვნის ენით დაწერილი დოკუმენტის წარდგინების აღსაწერად, მაგალითად HTML-ის გამოყენებით.[1] CSS არის მსოფლიო აბლაბუდის ტექნოლოგიური ქვაკუთხედი, HTML-თან და JavaScript-თან ერთად.

js

JavaScript (/ˈdʒɑːvəskrɪpt/), ხშირად შემოკლებით JS, არის პროგრამირების ენა და ქსელის ძირითადი ტექნოლოგია, HTML და CSS-თან ერთად. ვებგვერდების 99% იყენებს JavaScript-ს კლიენტის მხარეს ვებგვერდის ქცევისთვის.
